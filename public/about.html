About WikiNode

WikiNode is my final-year project that turns Wikipedia into an interactive visualisation, showing the hidden connections between articles. Instead of following info in a boring, linear way, it lets you explore topics in a network.

Project Overview:
The aim is to create an intuitive tool that helps understand how Wikipedia articles relate to one another using modern web tech. It uses the Wikipedia REST API and graph visualisation techniques to make exploring topics easy.

Motivation:
I built this project out of a genuine passion for data visualisation and graph theory. It shows off not just my technical skills, but also my creative side when it comes to presenting complex data in a way that’s engaging and educational.

Future Directions:
Looking ahead, I plan to improve performance, add more interactive features and analytical tools, and possibly work with other datasets. This project is just the start of even bigger visualisation ideas.

About the Developer:
I’m a final-year student with a real interest in web development and data visualisation. This project reflects both my academic journey and my love for finding new ways to make complex data accessible and fun.
